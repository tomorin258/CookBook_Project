# RecipeApp

一个基于 JavaFX 的食谱管理应用。

## 目录结构
```
├── src/main/java
│   ├── dao/
│   ├── service/
│   ├── init/
│   ├── model/
│   ├── data/
│   ├── view/
│   └── controller/
├── src/main/resources/
│   ├── fxml/
│   └── images/
├── config/
│   ├── mybatis_config.xml
│   └── MyBatisUtil.java
├── pom.xml
├── CookBook.sql
└── README.md
```
## 依赖
- Java 17+
- JavaFX
- Maven
