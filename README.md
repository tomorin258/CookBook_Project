# RecipeApp

一个基于 JavaFX 的食谱管理应用。

## 目录结构

```
├── src/main/java
│   ├── model/
│   ├── dao/
│   ├── init/
│   ├── data/
│   ├── view/
│   └── controller/
├── src/main/resources/
│   ├── fxml/
│   ├── mybatis_config.xml
│   └── images/
├── pom.xml
└── README.md
```

## 依赖
- Java 17+
- JavaFX
- Maven
